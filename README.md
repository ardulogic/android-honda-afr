# Bluetooth App for controling AFR for Spartan V2

## What you need
- Spartan v2 with a Bosch wideband o2 sensor
- Cheap bluetooth OBD2 scanner
- Arduino (or other controller) to connect to serial from Spartan and relay it via bluetooth 

## What is this for?
- View the AFR ratio in a realtime graph
- Control the AFR ratio target from the app
- View main OBD2 data points (Coolant temp, Short/Long Fuel Trim, MAP, RPM...)
- Calculate fuel consumption in real time
- Track fuel consumption (instant, trip, total)
